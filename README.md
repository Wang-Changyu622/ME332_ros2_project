# ROS2 移动机械臂控制系统

## 项目概述

本项目基于 ROS2 开发了一个集成自主导航、机械臂操作与多模态人机交互的移动机械臂系统。系统在 Gazebo 仿真环境中实现了同步定位与建图（SLAM）、自主导航与避障、机械臂运动控制、手势识别与语音控制等功能，体现了“感知-决策-控制-交互”的完整机器人系统闭环。

## 主要功能

- **实时同步定位与建图（SLAM）**：基于 Cartographer 实现，构建精确的 2D 环境地图。
- **自主导航与动态避障**：基于 Nav2 导航栈，采用 MPPI 控制器进行平滑路径规划与避障。
- **机械臂运动规划与控制**：基于 MoveIt! 框架实现六自由度机械臂的运动规划与抓取操作。
- **多模态人机交互**：
  - 手势识别：基于 MediaPipe 实现手部关键点检测与手势控制。
  - 语音控制：基于 VOSK 离线识别引擎，支持中英文指令。
- **多模态融合决策**：通过优先级仲裁策略融合手势与语音指令，保证控制一致性与安全性。

## 系统架构

系统采用分层模块化设计：

1. **感知层**：激光雷达、RGB-D 相机、IMU 等传感器。
2. **决策规划层**：Cartographer（建图）、Nav2（导航）、MoveIt!（机械臂规划）、多模态融合模块。
3. **控制执行层**：机器人控制器管理器、Gazebo 仿真器。
4. **人机交互层**：手势识别节点、语音控制节点。

## 技术实现

- **SLAM**：Cartographer 基于图的优化方法，支持多传感器融合与闭环检测。
- **导航**：Nav2 + MPPI 控制器，长视距预测、随机采样优化，适应动态环境。
- **机械臂控制**：MoveIt! + OMPL 规划器 + KDL 逆运动学求解器。
- **手势识别**：MediaPipe 手部 21 关键点检测，支持多帧验证。
- **语音控制**：VOSK 离线识别，支持自定义指令集。
- **系统集成**：ROS 2 节点-话题-服务架构，Launch 文件层次化启动管理。
